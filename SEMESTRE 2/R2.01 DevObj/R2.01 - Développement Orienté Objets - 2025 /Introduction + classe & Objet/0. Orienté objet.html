Orienté objet, classes, objets et modélisation

Orienté objet

L'orienté objet est une approche consistant à lier des données et des traitements dans des objets. Un objet est une entité qui possède des caractéristiques (des données) et des comportements (des traitements).

Programmation orienté objet

La programmation orienté objet est une approche de programmation qui consiste à créer des objets et à les faire interagir entre eux. Il existe des langages de programmation qui sont dits orientés objet (Java, C++, Python, PHP, ...).

Remarque : il est possible de faire de la programmation orienté objet dans des langages qui ne sont pas orientés objet (C, Fortran, ...) mais on ne peut pas mettre en place (facilement) certains concept de la programmation orienté objet (héritage, polymorphisme, ...).

Pour bien comprendre l'intérêt de ce type de programmation, imaginez que vous voulez créer un jeu vidéo. Dans ce jeu, il y aura des entités (nommés objet en programmation) tels que des joueurs, des places, des armes, etc.

Ces objets sont définis par :

  • Des caractéristiques (nommées attributs) : ce sont les données qui décrivent l'objet.

Exemple : Un "joueur" a un nom, un nombre de point de vie, un niveau etc.

  • Des comportements (nommés méthodes) : ce sont les actions que l'objet peut effectuer.

Exemple : Un "joueur" peut changer de zone, attaquer, ou prendre un objet.

Ces objets interagissent entre eux pour résoudre des problèmes ou accomplir des tâches.

Exemple : Un joueur peut agir sur un autre joueur (par exemple l'attaquer), il peut changer de zone (interaction avec l'objet zone), prendre un objet (interaction avec un objet).

L'idée est de pouvoir modéliser ces objets en programmation pour pouvoir les faire intéragir entre eux.

Classes

Les langages orienté objet permettent de définir des classes. Une classe est un modèle qui permet de créer des objets. Une classe définit les caractéristiques et les comportements des objets qui en sont issus. Un exemple de classe est donné ci-dessous :

class Joueur
    nom
    pointsDeVie
    niveau
    attaquer()
    soigner()
    mourir()
    changerZone()

Objets

Un objet est une instance d'une classe. Un objet est une entité qui possède des caractéristiques et des comportements. Un objet est créé à partir d'une classe. Un exemple d'objet est donné ci-dessous :

Joueur1 : Joueur
    Raoul
    100
    3
    attaquer()
    soigner()
    mourir()
    changerZone()

Plusieurs objets peuvent instancier une même classe :

Joueur2 : Joueur
    Maya
    60
    10
    attaquer()
    soigner()
    mourir()
    changerZone()

Modélisation

La modélisation est une étape importante dans le développement d'un logiciel. Elle permet de définir les classes et les objets qui seront utilisés dans le logiciel. Elle permet aussi de définir les relations entre les classes et les objets.

La modélisation se fait en dehors d'un langage de programmation. Elle est faite dans un langage de modélisation. Il existe plusieurs langages de modélisation (UML, Merise, ...). Nous utiliserons le langage UML.
